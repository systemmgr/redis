## redis  
  
redis is a software project that implements data structure servers  
  
Automatic install/update:

```shell
bash -c "$(curl -LSs https://github.com/systemmgr/redis/raw/main/install.sh)"
```

Manual install:
  
requires:

Debian based:

```shell
apt install redis
```  

Fedora Based:

```shell
yum install redis
```  

Arch Based:

```shell
pacman -S redis
```  

MacOS:  

```shell
brew install redis
```
